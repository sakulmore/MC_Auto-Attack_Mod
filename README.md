# Minecraft (Fabric) - Auto Attack Mod
Auto Attack is a quality-of-life and accessibility combat mod designed for Minecraft Java Edition 26.2 (Chaos Cubed). It eliminates the need for spam-clicking during combat. Instead of repeatedly clicking the Left Mouse Button (LMB), you can simply hold it down. The mod will automatically calculate your weapon's cooldown and strike exactly when your attack strength is fully charged.

This project is an updated remake of the original "Auto Attack" mod created by vin350, specifically rewritten for the latest version of the game.
